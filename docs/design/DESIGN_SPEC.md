## 1. Design Philosophy

- Minimal
- Developer-focused
- Content-first
- No visual noise

Design prioritizes:
- Writing efficiency (Admin)
- Reading comfort (Public)

---

## 2. Admin UI Design

### Layout
- Dark navbar
- Light content area
- Two-column editor (8 / 4 grid)

### Typography
- System font stack
- Large title input
- Clear visual hierarchy

### Components
- Cards for meta boxes
- Minimal buttons
- No unnecessary icons

---

## 3. Public UI Design

### Layout
- Dark navbar
- Light background
- Centered content

### Post Cards
- Thumbnail on top
- Title prominent
- Excerpt visible
- Category & tags shown

### Post Detail
- Max-width content (~760px)
- Large line-height
- Code blocks visually separated

---

## 4. Color Usage

- Dark gray for navigation
- White/light gray for content
- Blue for primary actions
- Avoid excessive gradients

---

## 5. Responsive Behavior

- Mobile-first
- Cards stack vertically
- Editor sidebar moves below content on small screens

---

## 6. Design Constraints

- Bootstrap only
- No custom UI frameworks
- No animation-heavy effects

---

## 7. Extension Notes

Allowed:
- Minor spacing adjustments
- Typography refinement

Avoid:
- Over-styling
- Trendy UI patterns that hurt readability
